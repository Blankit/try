一、使用默认参数

```
m = Prophet()  # 定义模型
m.fit(df)  # 拟合数据
future = m.make_future_dataframe(periods=1)  # 构建预测集
forecast = m.predict(future)  # 进行预测
plt.figure()
m.plot(forecast)
plt.show()
#计算RMSE
y_real = df.loc[not_nan[0]].y
y_predict = forecast.loc[not_nan[0]].yhat
rms = sqrt(mean_squared_error(y_real, y_predict))
print(rms)
```
